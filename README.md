# Automatisierung Tool

Eine Fullstack-Webanwendung zur digitalen Aufgaben- und Prozessverwaltung.

Das Projekt kombiniert ein Django REST Backend mit einem React/Vite Frontend. Es unterstützt Benutzerregistrierung, Login mit JWT, Aufgabenverwaltung, Kanban-Board mit Drag & Drop, Dashboard-Statistiken und Diagramme.

## Funktionen

- Benutzerregistrierung
- Login mit JWT-Authentifizierung
- Demo-Zugang / Gastmodus
- Aufgaben erstellen, bearbeiten und löschen
- Statusverwaltung: Offen, In Bearbeitung, Erledigt
- Prioritäten: Niedrig, Mittel, Hoch
- Fälligkeitsdatum und Überfällig-Markierung
- Kanban-Board mit Drag & Drop
- Listenansicht
- Suche, Filter und Sortierung
- Dashboard mit Statistiken
- Diagramme für Status und Prioritäten
- Deployment mit Apache, Gunicorn und HTTPS
